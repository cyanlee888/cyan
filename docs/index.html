Product Readout

综合判断

关于「重新开始课程」

当前继续学习用户数占比只有 2.6%，且口径仍是聚合估算。建议先用用户级 SQL 复核真实用户规模，再决定是否做明显按钮；短期可考虑轻量入口或复习提醒。

关于留资模块优先级

Words 支付效率最好，课程兼具支付和 D7 留存，DINO 更偏短期兴趣，FM 是流量入口但效率偏低。优先优化 Words / 课程承接，同时改善 FM 后续付费引导。

数据来源：神策事件分析、留存分析与漏斗分析导出文件。页面中的聚合人数不等同于全周期去重人数；支付转化人数样本较小，建议结合更长周期和订单金额复核。
